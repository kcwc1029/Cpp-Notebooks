## 補充：printf 進階格式控制

### 完整格式語法回顧

```
% [flags] [width] [.precision] [length] type
  ↑                                      ↑
 必要                                   必要
```

**格式的五個組成部分：**

| 部分         | 是否必要 | 說明                                          |
| ------------ | -------- | --------------------------------------------- |
| `%`          | ✅ 必要  | 格式指定符的起始標記                          |
| `flags`      | 選填     | 對齊、正負號、補零等旗標                      |
| `width`      | 選填     | 最少輸出的字元數                              |
| `.precision` | 選填     | 小數位數，或字串最多輸出字元數                |
| `length`     | 選填     | `l`（long）、`h`（short）、`L`（long double） |
| `type`       | ✅ 必要  | 資料型別（`d`、`f`、`c`、`s` 等）             |

### type：型別字元完整對照

| 型別字元  | 輸入型別       | 說明                        | 輸出範例              |
| --------- | -------------- | --------------------------- | --------------------- |
| `d` / `i` | `int`          | 有號十進位整數              | `42`、`-17`           |
| `u`       | `unsigned int` | 無號十進位整數              | `4294967295`          |
| `o`       | `int`          | 八進位整數（不含 `0` 前綴） | `52`（代表十進位 42） |
| `x`       | `int`          | 十六進位整數（小寫）        | `2a`                  |
| `X`       | `int`          | 十六進位整數（大寫）        | `2A`                  |
| `f`       | `double`       | 小數表示浮點數              | `3.141593`            |
| `e`       | `double`       | 科學記號（小寫 e）          | `3.141593e+00`        |
| `E`       | `double`       | 科學記號（大寫 E）          | `3.141593E+00`        |
| `g`       | `double`       | 自動選 `f` 或 `e`（較短者） | 視數值決定            |
| `G`       | `double`       | 自動選 `f` 或 `E`           | 視數值決定            |
| `c`       | `int`/`char`   | 單一字元                    | `A`                   |
| `s`       | `char *`       | 字串                        | `Hello`               |
| `p`       | 指標           | 記憶體位址（十六進位）      | `0x0061FF18`          |
| `%`       | —              | 輸出 `%` 本身               | `%`                   |

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    int    iN  = 60;
    double dV1 = 0.003;
    double dV2 = 0.00003850168;

    /* 整數的進位輸出 */
    printf("%d 的八進位：%o\n", iN, iN);       /* 74 */
    printf("%d 的十六進位（小）：%x\n", iN, iN); /* 3c */
    printf("%d 的十六進位（大）：%X\n", iN, iN); /* 3C */

    /* %% 輸出百分號本身 */
    printf("今日折扣：85%%\n");  /* 輸出 85%，不是格式符！*/

    /* 浮點數的 f、e、E、g 比較 */
    printf("\n--- dV1 = 0.003 ---\n");
    printf("%%f 格式：%f\n",   dV1);   /* 0.003000（預設 6 位小數）*/
    printf("%%e 格式：%e\n",   dV1);   /* 3.000000e-03 */
    printf("%%E 格式：%E\n",   dV1);   /* 3.000000E-03 */
    printf("%%g 格式：%g\n",   dV1);   /* 0.003（選 f，因小數點前4位不全為0）*/

    printf("\n--- dV2 = 0.00003850168 ---\n");
    printf("%%f 格式：%f\n",   dV2);   /* 0.000039 */
    printf("%%e 格式：%e\n",   dV2);   /* 3.850168e-05 */
    printf("%%g 格式：%g\n",   dV2);   /* 3.85017e-05（選 e，前4位全為0）*/
    printf("%%G 格式：%G\n",   dV2);   /* 3.85017E-05 */

    system("pause");
    return 0;
}
```

### flags：輸出調整旗標

| 旗標        | 說明                           | 範例    | 效果                    |
| ----------- | ------------------------------ | ------- | ----------------------- |
| `-`         | 靠左對齊（預設靠右）           | `%-10d` | `42        `            |
| `+`         | 強制顯示正負號                 | `%+d`   | `+42` 或 `-42`          |
| `0`         | 不足寬度時補零                 | `%010d` | `0000000042`            |
| ` `（空白） | 正數前補一個空白（與負號對齊） | `% d`   | ` 42`（負數顯示 `-42`） |
| `#`         | 顯示進位前綴或強制顯示小數點   | `%#x`   | `0x2a`；`%#o` → `052`   |

### width 與 .precision：寬度與精確度

**width（最少輸出字元數）：**

- 數值位數不足時，**靠右對齊，左側補空白**（預設）
- 搭配 `-` 旗標可改為靠左對齊
- 搭配 `0` 旗標可改為補零
- 若數值位數**超過** width，忽略 width 設定，完整輸出

**.precision（精確度）：**

| 搭配型別                     | 效果                               |
| ---------------------------- | ---------------------------------- |
| `%d`、`%i`、`%u`、`%o`、`%x` | 最少輸出的**數字位數**（不足補 0） |
| `%f`、`%e`、`%E`             | 小數點後輸出的**位數**（預設 6）   |
| `%g`、`%G`                   | 最多有效數字的**總位數**（預設 6） |
| `%s`                         | 字串最多輸出的**字元數**           |

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    int    iVal = 60;
    double dVal = 9453.59;

    /* 定位尺（方便對齊觀察）*/
    printf("123456789012345\n");

    /* ===== 整數的 width/precision ===== */
    printf("[%d]\n",     iVal);    /* [60]           無控制 */
    printf("[%6d]\n",    iVal);    /* [    60]        width=6，靠右 */
    printf("[%-6d]\n",   iVal);    /* [60    ]        靠左 */
    printf("[%06d]\n",   iVal);    /* [000060]        補零 */
    printf("[%+6d]\n",   iVal);    /* [   +60]        顯示正號 */
    printf("[%6.4d]\n",  iVal);    /* [  0060]        precision=4，補至4位數 */
    printf("[%#6o]\n",   iVal);    /* [   074]        八進位加 0 前綴 */
    printf("[%#6x]\n",   iVal);    /* [  0x3c]        十六進位加 0x 前綴 */

    printf("\n");

    /* ===== 浮點數的 width/precision ===== */
    printf("[%f]\n",      dVal);   /* [9453.590000]   預設 6 位小數 */
    printf("[%e]\n",      dVal);   /* [9.453590e+03]  科學記號 */
    printf("[%4.2f]\n",   dVal);   /* [9453.59]       width<實際寬度，忽略 */
    printf("[%10.1f]\n",  dVal);   /* [    9453.6]    width=10，1位小數 */
    printf("[%010.3f]\n", dVal);   /* [009453.590]    補零，3位小數 */
    printf("[%15.4e]\n",  dVal);   /* [   9.4536e+03] width=15，科學記號4位 */
    printf("[%+10.2f]\n", dVal);   /* [  +9453.59]    顯示正號 */
    printf("[%-10.2f|\n", dVal);   /* [9453.59   |    靠左對齊 */

    printf("\n");

    /* ===== 字串的 precision ===== */
    char name[] = "Butterfly";
    printf("[%s]\n",      name);   /* [Butterfly]     完整輸出 */
    printf("[%12s]\n",    name);   /* [   Butterfly]  靠右，width=12 */
    printf("[%-12s]\n",   name);   /* [Butterfly   ]  靠左 */
    printf("[%.5s]\n",    name);   /* [Butte]         只輸出前5字元 */
    printf("[%12.5s]\n",  name);   /* [       Butte]  靠右+前5字元 */

    system("pause");
    return 0;
}
```

### length 修飾符

| 修飾符        | 說明        | 用途範例                       |
| ------------- | ----------- | ------------------------------ |
| `l`（小寫 L） | long        | `%ld`（long）、`%lf`（double） |
| `ll`          | long long   | `%lld`（long long）            |
| `h`           | short       | `%hd`（short）                 |
| `L`           | long double | `%Lf`（long double）           |

```c
long      lv  = 1234567890L;
long long llv = 9876543210123LL;
short     sv  = 32767;

printf("long       = %ld\n",  lv);    /* 1234567890 */
printf("long long  = %lld\n", llv);   /* 9876543210123 */
printf("short      = %hd\n",  sv);    /* 32767 */
```

### 實用格式組合速查表

| 需求             | 格式      | 輸入值      | 輸出         |
| ---------------- | --------- | ----------- | ------------ |
| 整數靠右，寬 8   | `%8d`     | 42          | `      42`   |
| 整數靠左，寬 8   | `%-8d`    | 42          | `42      `   |
| 整數補零，寬 8   | `%08d`    | 42          | `00000042`   |
| 顯示正負號       | `%+d`     | 42          | `+42`        |
| 浮點 2 位小數    | `%.2f`    | 3.14159     | `3.14`       |
| 浮點寬 10，2 位  | `%10.2f`  | 3.14159     | `      3.14` |
| 浮點補零         | `%010.2f` | 3.14159     | `0000003.14` |
| 科學記號 2 位    | `%.2e`    | 0.00314     | `3.14e-03`   |
| 字串最多 5 字元  | `%.5s`    | "Butterfly" | `Butte`      |
| 顯示十六進位前綴 | `%#x`     | 255         | `0xff`       |
