# 第十一章：參考 — 實作練習題目

---

## 基礎題

### 題目 1【宣告與修改】⭐

宣告一個整數變數 `score = 60`，建立它的參考 `ref_score`。

1. 透過 `ref_score` 將值增加 40
2. 分別印出 `score` 和 `ref_score` 的值
3. 印出兩者的記憶體位址，驗證位址相同

**預期輸出：**
```
score    = 100
ref_score = 100
&score    = 0x...（某個位址）
&ref_score = 0x...（與 score 相同的位址）
```

---

### 題目 2【const 參考函數】⭐

寫一個函數 `void displayInfo(const string& name, const int& age)`，用 `const 參考`接收姓名和年齡，並印出：

```
姓名：Alice，年齡：20 歲
```

要求：
1. 函數內嘗試修改參數（確認編譯器會報錯，然後移除）
2. 在 `main` 中測試，傳入變數與字串字面值

---

### 題目 3【傳參考：遞增函數】⭐

寫一個函數 `void increment(int& n, int step = 1)`，將 `n` 增加 `step`。

在 `main` 中測試：
```cpp
int x = 10;
increment(x);       // x → 11
increment(x, 5);    // x → 16
increment(x, -3);   // x → 13
```

---

## 中階題

### 題目 4【多值回傳】⭐⭐

寫一個函數：

```cpp
void calcStats(const int arr[], int size,
               int& sum, double& avg, int& maxVal, int& minVal);
```

用參考同時計算並回傳整數陣列的**總和、平均、最大值、最小值**。

測試資料：`{5, 3, 8, 1, 9, 2, 7, 4, 6}`

**預期輸出：**
```
總和：45
平均：5.0
最大：9
最小：1
```

---

### 題目 5【就地字串處理】⭐⭐

寫以下兩個函數，都接受 `string&` 參數，直接修改原字串：

```cpp
void reverseStr(string& s);       // 反轉字串
void removeSpaces(string& s);     // 移除所有空白字元
```

在 `main` 中對 `"Hello World !"` 依序呼叫這兩個函數，印出每次操作後的結果。

---

### 題目 6【泡沫排序 + 傳參考交換】⭐⭐

實作泡沫排序法：

1. 寫 `void mySwap(int& a, int& b)`：用參考實作交換（不使用 `std::swap`）
2. 寫 `void bubbleSort(int arr[], int size)`：使用 `mySwap` 完成排序
3. 每次交換後印出目前陣列狀態

測試資料：`{64, 34, 25, 12, 22, 11, 90}`

---

### 題目 7【Range-for + 參考應用】⭐⭐

有整數陣列 `{-3, 7, -1, 5, -8, 2, -4, 6}`，用 range-based for 迴圈完成：

1. 將所有**負數**改為正數（取絕對值）— 需要用 `&`
2. 印出修改後的陣列
3. 用 `const auto&` 計算所有元素的總和
4. 找出最大值和最小值

## 進階題

### 題目 8【結構體與參考】⭐⭐⭐

定義結構體：
```cpp
struct Point { double x, y; };
```

寫以下函數：

| 函數簽名 | 功能 |
| -------- | ---- |
| `void move(Point& p, double dx, double dy)` | 移動座標 |
| `double distance(const Point& p1, const Point& p2)` | 計算兩點距離 |
| `Point& closer(Point& p1, Point& p2, const Point& origin)` | 回傳離原點較近的點之**參考** |

在 `main` 中：
1. 建立兩個點 `A(3, 4)` 和 `B(1, 1)`
2. 計算兩點距離
3. 找出較近的點，並將其 x、y 座標都設為 0

---

### 題目 9【自訂安全陣列】⭐⭐⭐

設計一個「安全陣列」類別，包含邊界檢查：

```cpp
class SafeArray {
    int data[100];
    int sz;
public:
    SafeArray(int n);                // 建構子，初始化 sz，全部設為 0
    int& at(int index);              // 回傳參考，含邊界檢查
    int size() const;                // 回傳大小
    void print() const;              // 印出所有元素
};
```

要求：
- `at()` 如果 index 超出範圍，印出錯誤訊息並回傳 `data[0]` 的參考（不崩潰）
- 在 `main` 中：建立大小為 5 的陣列，修改元素，讀取元素，測試越界存取

---

### 題目 10【CSV 字串解析器】⭐⭐⭐

寫一個函數：

```cpp
bool parseCSV(const string& line,
              string& name, int& score, string& grade);
```

解析格式為 `"姓名,分數,等第"` 的字串，將三個欄位分別存入對應的參考參數。函數成功時回傳 `true`，格式錯誤時回傳 `false`。

測試資料：
```
"Alice,92,A"   → name="Alice", score=92, grade="A"
"Bob,78,C"     → name="Bob",   score=78, grade="C"
"invalid"      → 解析失敗，回傳 false
```

在 `main` 中讀取多行輸入，直到使用者輸入空行為止，解析並印出結果。
